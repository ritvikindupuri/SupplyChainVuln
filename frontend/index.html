Scan a GitHub Repository

Enter a public GitHub repository. SecureChain fetches only the recent commit diff deltas and runs three security agents in parallel, then validates with Gemini + Claude.

How many commits should I scan?
  • Auto — SecureChain picks the right number of commits for you, based on how active the repo is (recommended)
  • 3 — just check the few newest changes
  • 5 — check a recent pull request or small batch of work
  • 8 — a normal check for a repo with regular activity
  • 10 — check further back for slow-moving repos that don't change often
Try: expressjs/express pallets/flask lodash/lodash
